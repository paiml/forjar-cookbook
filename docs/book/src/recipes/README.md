# Recipes

Recipes are organized by category:

- **Infrastructure** — Developer workstation, web server, database, monitoring, security
- **Linux Administration** — Cron, users, sysctl, logs, time sync, systemd, patching
- **Nix-Style** — Dev shells, toolchain pins, build sandboxes using pepita isolation
- **Rust Builds** — Release builds, musl static binaries, cross-compilation
- **Package Distribution** — .deb/.rpm build, private repos, fleet deploy
- **OpenTofu Patterns** — Saved plans, check blocks, lifecycle rules, testing DSL
